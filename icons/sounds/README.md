# Celebration Sounds Directory

This directory should contain celebration sounds for each theme completion.

## Required Sound Files

| Filename | Theme | Description |
|----------|-------|-------------|
| `warrior_fanfare.wav` | Warrior | Epic battle victory fanfare with horns and drums |
| `scholar_chime.wav` | Scholar | Magical mystical chime with wisdom bells |
| `wanderer_adventure.wav` | Wanderer | Journey completion horn with nature sounds |
| `underdog_triumph.wav` | Underdog | Triumphant victory with crowd cheering |
| `scientist_eureka.wav` | Scientist | Scientific discovery sound with bubbling and "eureka" effect |

## Sound Specifications

- **Format**: WAV (recommended) or MP3
- **Duration**: 2-5 seconds
- **Sample Rate**: 44100 Hz
- **Channels**: Stereo preferred, Mono acceptable
- **Volume**: Normalized to -3dB peak

## Sound Style Guidelines

### Warrior Theme
- Brass fanfares
- Drum rolls
- Sword clashing (subtle)
- Heroic orchestral swells

### Scholar Theme
- Soft magical chimes
- Turning pages
- Mystical wind
- Ethereal bells

### Wanderer Theme
- Nature sounds (birds, wind)
- Adventure horn
- Footsteps on path
- Map unrolling

### Underdog Theme
- Crowd cheering
- Victory bells
- Triumphant horns
- Uplifting crescendo

### Scientist Theme
- Bubbling liquids
- Electronic beeps
- "Eureka" moment chime
- Laboratory ambiance

## Usage

These sounds play when the user clicks on a theme completion celebration card.
The CelebrationAudioManager in `entitidex/celebration_audio.py` handles playback.
